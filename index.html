About

  TerraCiv is an Online Multiplayer 3D Real-Time Strategy Game built with the latest web technologies.

  It is powered by HTML5, WebGL and WebSockets, enabling the 3D and the real-time gaming on the modern web browsers.

  TerraCiv is compatible with Google Chrome, Safari, Firefox, Chromium as well as Google Chrome for Android / iOS.
